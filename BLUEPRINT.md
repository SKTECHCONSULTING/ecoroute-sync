## Database Schema
```sql
CREATE EXTENSION IF NOT EXISTS postgis;

CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  cognito_sub VARCHAR(255) UNIQUE NOT NULL,
  email VARCHAR(255) UNIQUE NOT NULL,
  full_name VARCHAR(255),
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE tasks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  requester_id UUID REFERENCES users(id),
  type VARCHAR(50) CHECK (type IN ('pickup', 'drop-off')),
  location GEOMETRY(Point, 4326) NOT NULL,
  description TEXT,
  status VARCHAR(50) DEFAULT 'pending',
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE trips (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  driver_id UUID REFERENCES users(id),
  start_location GEOMETRY(Point, 4326) NOT NULL,
  end_location GEOMETRY(Point, 4326) NOT NULL,
  planned_path GEOMETRY(LineString, 4326),
  departure_time TIMESTAMP NOT NULL,
  available_capacity INTEGER DEFAULT 1,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE matches (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  task_id UUID REFERENCES tasks(id),
  trip_id UUID REFERENCES trips(id),
  match_score FLOAT,
  ai_reasoning TEXT,
  status VARCHAR(50) DEFAULT 'suggested',
  created_at TIMESTAMP DEFAULT NOW()
);
```

## API Specification
- `POST /tasks`: Post a new logistics request (pickup/drop-off)
- `POST /trips`: Log a planned commute or trip for path-sharing
- `GET /matches`: Retrieve AI-optimized matches using Gemini 1.5 Flash
