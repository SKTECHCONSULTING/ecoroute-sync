# EcoRoute Sync

AI-orchestrated coordination engine for zero-emissions logistics.

## Project Structure

- `infra/`: AWS CDK infrastructure (Cognito, RDS PostGIS, Lambda)
- `backend/`: API handlers and Drizzle schema (TypeScript)
- `web/`: Next.js 14 web dashboard (Tailwind CSS)
- `mobile/`: Flutter mobile application (Dart)

## Stack

- **Framework:** Flutter (Mobile) + Next.js 14 (Web)
- **Language:** TypeScript (Backend) & Dart (Mobile)
- **Styling:** Tailwind CSS (Web)
- **Database:** PostgreSQL with PostGIS (AWS RDS/Aurora)
- **ORM:** Drizzle
- **Auth:** AWS Cognito
